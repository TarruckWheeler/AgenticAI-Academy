## 📚 Learning Paths

### 🟢 Beginner: AI Foundations (80-100 hours)

- **Outcome:** Build AI agents without coding

### Quick Start

```bash
# Clone the repository
git clone https://github.com/AgenticAI/academy.git

# Install dependencies
cd academy
pip install -r requirements.txt

# Run locally
python app.py
```
